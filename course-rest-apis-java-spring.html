What You Will Learn

Java Basics

Object-oriented principles and essential syntax for real backend work.

RESTful APIs

What REST is, how APIs work, and how to design and implement them with Spring Boot.

Git Fundamentals

Version control basics to manage, track, and share your code efficiently.

JUnit Testing

How to write effective unit tests so you can refactor without fear.

Lombok

Reduce boilerplate (getters/setters/builders) with intention and good boundaries.

Maven

Dependency management and builds for Java projects.

Your IDE

Eclipse (course baseline), IntelliJ, or VS Code — setup and navigation.

Databases

Start simple, then connect a real database and integrate it with Spring.

Spellcraft mindset

You can use AI to move faster — but you stay in control by understanding the connections: HTTP, validation, errors, persistence, and tests.
